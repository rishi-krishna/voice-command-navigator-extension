Voice Command Navigator Chrome Extension

Navigate the web by voice.

Control tabs, scroll pages, and click links with simple voice commands. Built for accessibility, multitasking, and hands-free browsing.

Get the extension View commands

Command examples

Tabs

"next tab", "previous tab", "switch tab 3", "close tab"

Scrolling

"scroll down", "scroll up", "scroll to top", "scroll to bottom"

Links

"show links" then "click link 5"

Open sites

"open youtube" (opens or searches)

Install (developer mode)

  1. Open chrome://extensions in Chrome.
  2. Enable Developer mode.
  3. Click Load unpacked and select the extension folder.
  4. Open the extension and toggle Always-on voice (a small listener window opens).
