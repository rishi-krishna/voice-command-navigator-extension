# Voice Command Navigator (Chrome Extension)

Hands-free tab navigation, scrolling, and link clicking with voice commands.


## Commands
- "scroll down" / "scroll up" / "scroll to top" / "scroll to bottom"
- "show links" then "click link 3"
- "next tab" / "previous tab" / "switch tab 4"
- "close tab" / "open new tab" / "reload tab"
- "open youtube" (searches or opens a site)
- "go back" / "go forward" / "reopen last closed tab"
- "reopen last closed tab" restores the most recently closed tab
- "pause video" / "play video" / "mute tab" / "unmute tab"
- "zoom in" / "zoom out" / "zoom reset" / "print page" / "reader mode"
- Enable dictation to type into focused input fields
- "dictation on" / "dictation off"
